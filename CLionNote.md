# 记录个人在 CLion 中常用的快捷键


## 代码查看
- 快速查看类的函数定义 Ctrl+F12

		Main menu | Navigate | File Structure

- 切换到 .h / .cpp （已修改快捷键 Alt+O）

		Main menu | Navigate | Related Symbol

- 查看相关上下文 Alt+Q （查看位于什么函数或命名空间）

		View | Context Info

- 查看使用 Alt+F7

		Edit | Find | Find Usages



## 编辑
- 查看剪贴板历史 Ctrl+Shift+V

		Main menu | Edit | Paste from History...
		



## 调试，运行
- 编译 Ctrl+F9
- 调试 Shift+F9
- 运行调试版本 Shift+F10
